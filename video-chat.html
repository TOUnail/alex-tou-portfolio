This is a one-on-one video chat app similar to the public chat website Omegle. The only difference is, you don't pair up randomly with strangers. Instead, you call the person you want to chat with by giving them your provided unique ID.

In a sense, this app is a combination of Omegle and Zoom where you have a video chat with a friend by calling them or setting up a meeting.

How was it built?

This app was built with ReactJS using WebRTC and SocketIO

ReactJS

WebRTC

SocketIO

SocketIO is responsible for creating a private room for both users using the recipient's ID and the communication aspect of the app. WebRTC is used to display a video/audio stream to the end user.

Challanges

There were several Challanges in developing this app. The first one was beyond my control. The web API MediaDevices.getUserMedia() does not work on mobile devices without an SSL certificate. This issue was managable but it was an obstacle that hindered mobile debugging.
